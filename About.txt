
* 키보드 설명
  - 탭                          : 좌우 뒤집기
  - 엔터                        : 페이지 선택
  - `                           : 툴바/상태바 보이기 전환
  - 스페이스/화살표 아래/PG DN  : 다음 페이지
  - 화살표 위/PG UP             : 이전 페이지
  - 이스케이프/F                : 풀스크린 전환
  - F5                          : 화면 갱신
  - 홈                          : 맨 첫 페이지
  - 엔드                        : 끝 페이지
  - 2                           : 2페이지 보기 전환
  - 1                           : 1페이지 보기
  - ALT-O                       : 폴더 열기
  - CTL-O                       : 파일 열기
  - CTL-P                       : PCO/ZIP 파일 열기
  - S                           : 샤픈 전환
  - I                           : 인터폴레이션 전환
  - Z                           : 확대 전환


* 기타 
  - 마우스 가운데 버튼          : 풀스크린 전환
  - 파일 드롭                   : 파일 열기(GIF,JPG,BMP,PCO,ZIP)
  - 폴더 드롭(1개만 인식)       : 폴더 열기
  - 마우스 휠                   : 이전/다음 이미지 보기


* 필터 설명
  - 인터폴레이션 
    이미지 크기 변형시 점과 점사이의 값을 평균을 내에서 
    부드럽게 축소/확대하여 준다.

  - 샤픈
    이미지의 경계를 추출하여서 뿌연 이미지를 선명하게 
    만들어 준다.
